# Packers and Movers UI : https://mrnareshkumar.github.io/packers-movers-app/

ReactJS implementation for the Packers and Movers assignment.

## Features

- Fetches move data from `https://apis2.ccbp.in/packers-and-movers/packers-and-movers-details`
- Displays the "My Moves" list with source, destination, request number, move date, property size, item count, and distance
- Expands and collapses move details with the "View move details" button
- Renders additional information, house details, and inventory details from the API `items` object
- Expands and collapses inventory categories
- Includes loading and error states

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
